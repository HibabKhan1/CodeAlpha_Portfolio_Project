# CodeAlpha_Portfolio_Project
CodeAlpha_Portfolio_Project is a Python-based stock portfolio management tool with a user-friendly GUI built using Tkinter. It allows users to manage their stock investments by fetching real-time and historical stock data, providing portfolio summaries, and exporting data to CSV files.

## Overview
CodeAlpha_Portfolio_Project is a dynamic stock portfolio management tool developed using Python and Tkinter. It allows users to manage their stock investments by adding, removing, and tracking stock performance. The project fetches real-time and historical stock data using the `yfinance` library and provides a graphical user interface for easy interaction.

## Features
- **Real-time Stock Data:** Fetches the latest stock data for any given symbol.
- **Historical Stock Data:** Retrieves and displays historical stock data for a specified period.
- **Portfolio Management:** Add, remove, and view stocks in your portfolio.
- **Portfolio Summary:** Provides a detailed summary of your portfolio, including total investment, portfolio value, and unrealized profit/loss.
- **Export to CSV:** Export your portfolio data to a CSV file for further analysis.
- **Graphical User Interface (GUI):** User-friendly GUI built with Tkinter.
